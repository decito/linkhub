# Decito's Linkhub

This application serves as a centralized hub for all your social media profiles, allowing you to manage and share your links effortlessly.

## Features

- Simple interface
- Customizable links
- Real-time analytics
- Dark mode
- Social sharing
- Responsive design
- SEO-friendly
